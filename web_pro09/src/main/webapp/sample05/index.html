DOM 객체 선택/접근

document.getElementById

HTML 태그의 속성중 id 속성의 값으로 태그를 선택/접근 하기 위한 방법
id로 접근하는 경우 가장 먼저 선택한 요소 1개만 반환한다.

document.getElementsByTagName(tag_name)

HTML 태그 이름으로 선택/접근 하기 위한 방법
동일한 태그명에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getElementsByName(tag_name)

HTML 태그의 속성중 name 속성의 값으로 태그를 선택/접근 하기 위한 방법
동일한 name 속성 값에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getElementsByClassName(class_name)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
동일한 class 속성 값에 해당하는 모든 요소를 HTMLCollection으로 반환한다.

document.getSelector(css_selector)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
css 선택자를 선택자를 사용하여 id, 태그명, class 등의 속성을 활용할 수 있다.
가장 먼저 선택한 요소 1개만 반환한다.

document.getSelectorAll(css_selector)

HTML 태그의 속성중 class 속성의 값으로 태그를 선택/접근 하기 위한 방법
css 선택자를 선택자를 사용하여 id, 태그명, class 등의 속성을 활용할 수 있다.
동일한 css 선택자에 해당하는 모든 요소를 nodeCollection 으로 반환한다.

선택된 DOM 객체를 기준으로 다른 DOM 객체 선택/접근

선택된 DOM 객체로 부터 상대적인 방법으로 다른 DOm 객체에 접근하는 방법

  • HTMLElement.firstElementChild : 하위 요소 중 첫번째 요소
  • HTMLElement.lastElementChild : 하위 요소 중 마지막 요소
  • HTMLElement.previousElementSibling : 동일 레벨에 있는 이전 요소
  • HTMLElement.nextElementSibling : 동일 레벨에 있는 다음 요소
  • HTMLElement.parentElement : 부모(상위) 레벨의 요소
